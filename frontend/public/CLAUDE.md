# CLAUDE.md - frontend/public

## 역할
Next.js 정적 파일 제공 폴더. 빌드 과정 없이 루트 경로(`/`)에서 직접 서빙되는 파일을 보관한다.

## 파일 목록
| 파일명 | 역할 |
|--------|------|
| (정적 파일) | favicon, 이미지, robots.txt 등 정적 에셋 |

## 규칙
- 이 폴더의 파일은 URL에서 `/파일명`으로 직접 접근 가능
- 코드에서 참조 시 `/파일명` 절대 경로 사용 (next/image 권장)
- 소스 코드(.ts, .tsx)는 이 폴더에 두지 않는다

## 관련 폴더
- `../src/` — 애플리케이션 소스 코드
